Main menu

The Main menu is build from floated list elements and easily extendable. It invlude the top part (class="header-a") and the main menu (class="header-b")

Editing the Main menu
The main menu is created by using the items for mobile menu and for desktop menu. It's easy in case, when you want to display another menu on mobile devices.

1. Mobile menu class: 

<div class="mobile-menu">
	<nav>
		<ul>
			<li><a class="has-child" href="#">HOME</a>
				<ul>
					<li><a href="index.html">Home style one</a></li>
					<li><a href="index_02.html">Home style two</a></li>
					<li><a href="index_03.html">Home style three</a></li>
					<li><a href="index_04.html">Home style four</a></li>	
				</ul>						
			</li>
			<li><a href="#">Contact Us</a></li>
		</ul>
	</nav>
</div>
2. Desktop menu class: 
<nav class="header-nav">
	<ul>
		<li><a href="#">Home</a>
			<ul>
				<li><a href="index.html">Home style one</a></li>
				<li><a href="index_02.html">Home style two</a></li>
				<li><a href="index_03.html">Home style three</a></li>
				<li><a href="index_04.html">Home style four</a></li>	
			</ul>
		</li>
		<li><a href="#">Hotels</a>
			<ul>
				<li><a href="hotel_list.html">Hotels standard list</a></li>
				<li><a href="hotel_simple_style.html">Hotels simple style</a></li>
				<li><a href="hotel_detail_style.html">Hotels detail style</a></li>
				<li><a href="hotel_detail.html">Hotel item page</a></li>
				<li><a href="hotel_booking.html">Hotel booking page</a></li>
				<li><a href="booking_complete.html">booking complete page</a></li>
			</ul>
		</li>
	</ul>
</nav>
You can easily edit the Main menu item by replacing or editing the text between the <a> element. Editing the navigation will extend the bar to the left, because they is positioned to the right. To set the navigation point related to the page on the status active u must add the class active to the related <li> element.

Main menu has dropdown submenu. You can easily add it to any Main menu element just adding submenu ul-li stucture.

<nav>
	<ul>
		<li><a class="has-child" href="#">HOME</a>
			<ul>
				<li><a href="index.html">Home style one</a></li>
				<li><a href="index_02.html">Home style two</a></li>
				<li><a href="index_03.html">Home style three</a></li>
				<li><a href="index_04.html">Home style four</a></li>	
			</ul>						
		</li>
		<li><a href="#">Contact Us</a></li>
	</ul>
</nav>

C) JavaScript - top

1. BxSlider
2 .prettyPhoto
3. jquery.customSelect()
4. Isotope - Filter & sort magical layouts
5. jQuery appear plugin
6. BxSlider
7 .jQuery Form Styler
8 .jQuery UI
9 .jQuery scroroller Plugin

$notweets = '2';

$consumersecret = 'owrII6Imsw4B1YzOKsmxleV9HMI2qfjSaSANkFyY2T8';
$accesstoken = '234541855-BKM0TaVI6Ns4uX3CgLdtY75C5n6FANSmltyPSQZN';
